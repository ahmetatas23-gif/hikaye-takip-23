📖 Metin Görüntüleme Ayarları

Öğrenci teste başladığında metin nasıl gösterilsin?

İpucu: "Sadece İlk Sayfada" modu öğrencilerin metni dikkatlice okumalarını sağlar. "Her Soruyla Birlikte" modu ise öğrencilerin metne sürekli bakabilmelerini sağlar.

JSON Format Örneği:

{
  "title": "Türkçe Okuma Anlama Testi",
  "description": "2. Sınıf Türkçe Okuma Anlama Testi",
  "readingText": "BURAYA OKUMA METNİ GELECEK",
  "textDisplayMode": "first_page_only",
  "questions": [
    {
      "type": "multiple_choice",
      "question": "SORU METNİ",
      "options": ["A) ŞIK 1", "B) ŞIK 2", "C) ŞIK 3"],
      "correct": 0,
      "points": 5,
      "timeLimit": 75
    },
    {
      "type": "fill_blank",
      "question": "SORU METNİ",
      "options": ["cevap1", "cevap2", "cevap3"],
      "correct": "cevap1",
      "points": 5,
      "timeLimit": 75
    },
    {
      "type": "true_false",
      "question": "SORU METNİ",
      "correct": true,
      "points": 5,
      "timeLimit": 75
    }
  ]
}

Soru Tipleri:

  • multiple_choice: Çoktan seçmeli (options dizisi ve correct index gerekli)
  • fill_blank: Boşluk doldurma (correct string gerekli)
  • true_false: Doğru/Yanlış (correct boolean gerekli)

Soru Ayarları:

  • points: Her soru için puan değeri (1-20 arası, varsayılan: 5)
  • timeLimit: Her soru için süre limiti saniye cinsinden (30-300 arası, varsayılan: 75)

Metin Görüntüleme Modu:

  • textDisplayMode: "first_page_only" (varsayılan), "with_questions", "no_text"
  • first_page_only: Önce metin, sonra sorular
  • with_questions: Her soruda metin görünür
  • no_text: Metin hiç gösterilmez

💡 İpucu: Düzenleme modunda her sorunun puan değeri ve süre limiti ayrı ayrı ayarlanabilir!
